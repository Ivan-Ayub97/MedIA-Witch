About MedIA-Witch

MedIA-Witch is a powerful open-source desktop application for enhancing images and videos using advanced AI models. Built upon the QualityScaler framework, it includes optimized models such as Real-ESRGAN, SRGAN, BSRGAN, and IRCNN, providing an intuitive interface for improving your media quality and resolution.

üöÄ Installation

Follow these simple steps to install MedIA-Witch:

  1. Run the installer and follow the on-screen instructions.
  2. Launch the application by opening MedIA-Witch.exe on your Windows system.
  3. Start enhancing media with just a few clicks!

MedIA-Witch uses PyInstaller and Inno Setup for easy installation.

üõ†Ô∏è How to Use

  1. Launch the application: Run MedIA-Witch.exe as administrator for optimal performance.
  2. Upload files: Add images, videos, or folders for enhancement.
  3. Set configuration: Choose an AI model and adjust output settings (resolution, format, etc.).
  4. Start the enhancement: Click Start and let MedIA-Witch enhance your media.
  5. Save the results: After processing, the enhanced files will be saved in your chosen location.
