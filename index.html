About MedIA-Witch

MedIA-Witch is a powerful open-source desktop application designed to enhance images and videos using advanced AI models. Built upon QualityScaler, the tool integrates optimized models like Real-ESRGAN, SRGAN, BSRGAN, and IRCNN.

Key Features

  • Advanced AI Models for Quality Enhancement: Real-ESRGAN, SRGAN, BSRGAN, and IRCNN for improving media quality.
  • Batch Processing: Process multiple images and videos simultaneously.
  • Customizable Settings: Adjust resolution, format, and AI model for the best enhancement results.
  • User-Friendly Interface: Easy to use for users of all levels.
  • Open-Source: Released under the MIT License.

Installation

  1. Run the installer and follow the instructions.
  2. Launch the application: Open MedIA-Witch.exe.
  3. Start enhancing your media!

How to Use

  1. Launch the application as administrator.
  2. Select your media files: Upload images or videos.
  3. Configure your settings: Choose AI model, resolution, and format.
  4. Start the enhancement process by clicking "UPSCALE".
  5. Save your files once processing is complete.

System Requirements

  • Operating System: Windows 10 or later.
  • RAM: Minimum 4GB, recommended 8GB.
  • GPU: NVIDIA GPU for better performance.
  • Storage: Space for processed files.
